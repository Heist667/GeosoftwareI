First assignment, Thomas Kujawa

This assignment takes two different datas, called route.js and polygon.js. It has to show different sections between a given route running through a given polygon. The sections should be sorted by descending length. Both, start and end points of the sections have to be shown. The last column should make clear, if the section is inside or outside the polygon. The lengths are rounded to three decimals places.

Sequence Length (in meter) Position to polygon Start-point (longitude, latitude) End-point (longitude, latitude)
